Alaric Hamacher
version 1.0.0
http://lab3d.kw.ac.kr

1. Create 2D Subtitles:

Create a subtitle track as you would in 2D. Render only the subtitle track to a movie with alpha. If it is a 3D project render only one eye and make sure to check burn in option for subtitles. Name the exported movie subtitles.mov .

2. Reimport the 2D Subtitle Movie with Alpha

Import the asset subtitle.mov file. Convert the 2D file to stereoscopic 3D with the context menu on right mouse click.

3. Generate 3D Subtitle EDL

Click Create EDL to create an EDL of the first subtitle track. Copy the result and save it to a file subtitles.edl .

4. Import the 3D Subtitle EDL:

Create a new Timline using import EDL. The Timline will have the exact cuts and positions of the subtitles. Select all the clips on the timeline and copy them to a new track on the top of your project timeline.

5. Adjust Subtitle Convergence Values:

Position your subtitles. Apply changes in convergence. You can also use keyframes to create animations.

6. Generate a DCDM sterescopic subtitle file:

Retrieve the convergence values and keyframes with the button create 3D subs. Copy and modify your subtitles for your further mastering process.
